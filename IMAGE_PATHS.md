# Available Image Paths for Admin Panel

## How to Use Images in Admin Panel

### Option 1: Use Existing Images
Copy and paste one of these paths into the "Product Image" field:

**Featured Products:**
- `assets/img/featured1.png`
- `assets/img/featured2.png`
- `assets/img/featured3.png`

**Regular Products:**
- `assets/img/product1.png`
- `assets/img/product2.png`
- `assets/img/product3.png`
- `assets/img/product4.png`
- `assets/img/product5.png`

**New Arrivals:**
- `assets/img/new1.png`
- `assets/img/new2.png`
- `assets/img/new3.png`
- `assets/img/new4.png`

### Option 2: Upload New Images
1. Click "Choose File" button in the admin panel
2. Select your image file (JPG, PNG, GIF supported)
3. The image will be uploaded and the path will be automatically filled
4. Uploaded images will be stored in `uploads/` folder

### Image Requirements
- **Format:** JPG, PNG, GIF
- **Size:** Maximum 5MB
- **Recommended dimensions:** 400x400px for best display
- **Background:** Transparent or white background recommended

### Tips
- Use descriptive filenames for uploaded images
- Keep image file sizes reasonable for faster loading
- Test your product after adding to ensure image displays correctly
